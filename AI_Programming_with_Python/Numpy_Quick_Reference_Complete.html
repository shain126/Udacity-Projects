NumPy Quick Reference — Complete

This page includes the functions you asked for, grouped by category. Each function links to the official NumPy documentation (opens in a new tab).

General Purpose

Function / MethodDescription
numpy.ndarray.dtypeReturn the data-type of the elements of the array. Arrays are homogeneous; dtype describes element type.
numpy.ndarray.ndimReturn the number of array dimensions (rank). E.g., returns 2 for a 4×3 array.
numpy.ndarray.shapeReturn a tuple representing the array dimensions, e.g., (rows, columns) for a 2‑D array.
numpy.ndarray.sizeReturn the number of elements present in the array (equal to the product of the shape dimensions).
numpy.saveSave an array to a binary .npy file.
numpy.loadLoad arrays or pickled objects from .npy, .npz or pickled files.
numpy.random.randomReturn random floats in the half-open interval [0.0, 1.0) in the specified shape.
numpy.random.randintReturn random integers from the half-open interval [low, high) in a specified shape.
numpy.random.normalReturn random samples from a Gaussian (normal) distribution (specify mean, std, size).
numpy.random.permutationReturn a randomly permuted sequence (or permuted copy of an array).
numpy.reshape / numpy.ndarray.reshapeReturns an array containing the same elements with a new shape, without affecting the original array (method and function versions available).

Array Creation

Function / MethodDescription
numpy.onesReturn a new array of given shape and type, filled with 1s.
numpy.zerosReturn a new array of given shape and type, filled with 0s.
numpy.fullReturn a new array of given shape and type, filled with a specified value.
numpy.eyeReturn a 2‑D array with 1s on the diagonal and 0s elsewhere (identity matrix).
numpy.diagExtract the diagonal elements or construct a diagonal array from given values.
numpy.uniqueReturn the sorted unique elements of an array.
numpy.arrayCreate an n‑dimensional array from Python sequences or other array-like objects.
numpy.arangeReturn evenly spaced values within a half-open interval [start, stop).
numpy.linspaceReturn evenly spaced numbers over a specified closed interval [start, stop] (configurable).
numpy.ndarray.copyReturn a copy of the array (deep copy of the data).

Operating with Elements and Indices

Function / MethodDescription
numpy.insertInsert values along the given axis before the specified indices.
numpy.deleteReturn a new array after deleting sub‑arrays along a specified axis.
numpy.appendAppend values to the end of an array (creates a new array).
numpy.hstackStack arrays in sequence horizontally (column-wise).
numpy.vstackStack arrays in sequence vertically (row-wise); result is at least 2‑D.
numpy.sortReturn a sorted copy of an array.
numpy.ndarray.sortSort an array in‑place (method).

Set Operations

Function / MethodDescription
numpy.intersect1dFind the intersection of two arrays (unique, sorted).
numpy.setdiff1dFind the set difference of two arrays (elements in a not in b).
numpy.union1dReturn the unique, sorted array of values that are in either input array.
